# grep-archive
Grep archive search for string in any files on the filesystem, in archive and even inner archive.

Supported archive format are : 
- Tar Formats
  - tar.gz 
  - tar 
  - tgz 
  - tar.bz2 
  - tar.zst
- Zip Format
  - jar
  - ear 
  - war
